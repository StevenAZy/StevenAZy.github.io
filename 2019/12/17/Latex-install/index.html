Latex_install

Posted on 2019-12-17

之前就听人说LaTeX如何如何的好，甩word几条街…。博主不以未然，感觉Word挺不错的，但是在写毕业论文的时候，Word的排版以及公式的书写是真的很糟糕！
于是就尝试了一下LaTex，果然没有人能逃过哲学家王竟泽的真香定律。当时只是简单的试用了一下，如今博主换了电脑，决定要配置一些必要的东西，安装LaTex的时候，搜了好多的教程，但是很多的教程写的都比较模糊，不够详细，于是就想自己写一个比较适合新手用的教程。

首先，先介绍一下什么是LaTeX:

  1. LaTeX是一类用于编辑和排版的软件，用于生成PDF文档。
  2. LaTeX编辑和排版的核心思想在于，通过\section和\paragraph等语句，规定了每一句话在文章中所从属的层次，从而极大方便了对各个层次批量处理。
  3. LaTeX在使用体验方面，最不易被Word替代的四个方面：方便美观的数学公式编辑、不会乱动的退格对齐、非所见即所得因此可以在编辑的时候使用退格和换行整理思路但生成的PDF不影响美观、部分导师和刊物不接受Word排版的文章。

总之，在写文章方面，LaTeX要优于Word。

LaTeX的官网是https://www.latex-project.org，
打开之后是这个样子的。

点击Get，就可以直接跳转到下载页面，往下拉一下就可以看到如下页面。

然后根据你的操作系统选择相应的下载链接，博主的系统是Windows10的，所以我就主要提一下Windows下的安装方法。可以看到，Windows下有三个链接MikTex、proTeXt和TexLive，其实这些都是LaTeX的发行版本，主要就是看你自己选择什么了，通过不断的查询资料，博主最终选择的是MikTeX，点击进去之后界面是这样的。

进去之后直接点击Downloads.

直接下载Installer就可以了，速度还是挺快的,这个网址应该没有被墙。

下载完之后就可以直接安装了，这是个傻瓜安装，中间换一下安装路径就可以了，如果你的C盘足够大的话，你也可以不用换。
安装完成之后，开始文件夹中会出现如下的选项,第二个就是用来编辑和排版的编辑器。

LaTeX的安装到这儿就先告一段落，后面会专门讲一下如何使用LaTeX的教程。
