A New Active Labeling Learning Method for Deep Learning

Address

https://ieeexplore.ieee.org/document/6889457

Abstract

  1. Active labeling in deep learning aims at achieving the best learning result with a limited labeled dataset
  2. a new active labeling method, AL-DL, for cost-effective selection of data to be labeled
  3. three metrics for data selection: least confidence, margin sampling and entropy
  4. building blocks: stacked restricted Boltzmann machines and stacked autoencoders

Proposed method

  • AL-DL, a new algorithm for active labeling in deep learning, achieves high classfication accuracy using as few labeled samples as possible
  • AL-DL’s process:
    1. select the samples in the unlabeled pool that are most ambiguous or uncertain for the current model
    2. the newly labeled samples are added to the labeled pool and used to retrain the model
